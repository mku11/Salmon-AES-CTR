Build
To build the app you will need:  
1. Microsoft Visual Studio 2022  
2. .Net Multi-platform App UI development (part of the Visual Studio installer)
Note: Use "Fast Deployment" under Android project settings if you're debugging.

Package:
To package the app click on Build/Archive in Visual Studio.
