Detailed Description

A virtual file backed by an encrypted file.

Supports operations for retrieving AesStream for reading/decrypting and writing/encrypting contents.

◆ set_allow_overwrite()

set_allow_overwrite ( self,
bool value )

Warning! Allow overwriting on a current stream.

Overwriting is not a good idea because it will re-use the same IV. This is not recommended if you use the stream on storing files or generally data if prior version can be inspected by others. You should only use this setting for initial encryption with parallel streams and not for overwriting!

Parameters
valueTrue to allow overwriting operations
