Class: WSFileStream

WSFileStream(file, mode)

File stream implementation for Web Service files. This class can be used for random file access of remote files.

Constructor

new WSFileStream(file, mode)

Construct a file stream from an WSFile. This will create a wrapper stream that will route read() and write() to the FileChannel
Parameters:
Name Type Description
file WSFile The WSFile that will be used to get the read/write stream
mode string The mode "r" for read "rw" for write
