Class ReadableStreamWrapper

ReadableStream wrapper for RandomAccessStream. Use this class to wrap any RandomAccessStream to a JavaScript ReadableStream to use with 3rd party libraries.

        stream: RandomAccessStream,
        buffersCount?: number,
        bufferSize?: number,
        backOffset?: number,
        alignSize?: number,
    ): any

    Creates a native ReadableStream from a RandomAccessStream

    Parameters

    • stream: RandomAccessStream

      The source stream.

    • buffersCount: number = 1

      Number of buffers to use.

    • bufferSize: number = 524288

      The length of each buffer.

    • backOffset: number = 32768

      The backwards offset. Some media libraries might request data rewinding the stream just a few bytes backwards. This ensures those bytes are included so we don't reset the stream.

    • alignSize: number = 0

      The align size. Set to a positive number to override the stream aligned size.

    Returns any
